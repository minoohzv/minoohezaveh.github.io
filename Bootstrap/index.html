About Bootstrap

Bootstrap is one of the popular open-source CSS frameworks that allow you to build fast responsive websites. Responsive attributes take care of the layout for different screen sizes such as mobile, tablet, and desktop.

Simple Integration Great Grid System Good Documentation

Simple Integration

Bootstrap can be easily used with other frameworks and platforms such as "Angular" and "React". It allows you to create a completely new website or add its elements to the existing website and CSS file.

Dropdowns

Bootstrap's dropdown menu is a predefined menu that allows the user to select their desired value from the list. Bootstrap's dropdown menu offers different directions menus, including Dropup, Dropright, and Dropleft.

We are able to turn any button or link to a dropdown menu by adding <div class="dropdown"> and a few changes to the button's or link's definitions. You are also able to use different colors and sizes for your dropdown menu. Here is an example of how to turn a button into a dropdown menu:

  • Action
  • Another action
  • Something else here
<div class="dropdown">
              <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton1" data-bs-toggle="dropdown" aria-expanded="false">
                Dropdown button
              </button>
              <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton1">
                <li><a class="dropdown-item" href="#">Action</a></li>
                <li><a class="dropdown-item" href="#">Another action</a></li>
                <li><a class="dropdown-item" href="#">Something else here</a></li>
              </ul>
            </div>

Cards

Bootstrap's card is an excellent way to create flexible and extensible content containers. Bootstrap's card framework offers many useful and powerful features, including headers, footers, content, and background colors.

In order to use cards, you need to add <div class="card"> to your content. The content could be the mixture of any card class such as image, body, text, link, etc. The default value of cards' width is 100% but you are able to change them based on your need. Here is an example of how to use a card with an image and text:

Card title

Some quick example text to build on the card title and make up the bulk of the card's content.

Go somewhere 
<div class="card" style="width: 18rem;">
                <img src="..." class="card-img-top" alt="...">
                <div class="card-body">
                  <h5 class="card-title">Card title</h5>
                  <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
                  <a href="#" class="btn btn-primary">Go somewhere</a>
                </div>
              </div>

Navbar

Bootstrap's navbar is a responsive navigation header that can extend or collapse depending on the screen size. Bootstrap navbar offers many useful and powerful features, including branding, navigation, forms, color schemes, and more.

In order to use cards, you need to add <nav class="navbar navbar-expand{-sm|-md|-lg|-xl|-xxl"} > based on your preference of collapsing navigation header on different screen size. Here is an example of how to use a navbar with an image and text:

Navbar
  • Home
  • Features
  • Disabled 
<nav class="navbar navbar-expand-lg navbar-light bg-light">
                  <div class="container-fluid">
                    <a class="navbar-brand" href="#">Navbar</a>
                    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                      <span class="navbar-toggler-icon"></span>
                    </button>
                    <div class="collapse navbar-collapse" id="navbarNav">
                      <ul class="navbar-nav">
                        <li class="nav-item">
                          <a class="nav-link active" aria-current="page" href="#">Home</a>
                        </li>
                        <li class="nav-item">
                          <a class="nav-link" href="#">Features</a>
                        </li>
                        <li class="nav-item">
                          <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Disabled</a>
                        </li>
                      </ul>
                    </div>
                  </div>
                </nav>
